  What is SPARQL?  
Sparql Protocol And RDF Query Language

The internet is full of scientific and non-scientific databases with immense volumes of data ready available to users. Unfortunately,
such databases would be meaningless (or incredibly painful to use) without orderly, systematic methods of accessing them. SPARQL
is a Semantic Query Language - it provides intuitive, simplistic ways to access desired data from compatible databases, that
is, databases with a SPARQL endpoint.

Specifically, SPARQL is able to extract data stored in the Resource Database Framework (RDF) format. The core idea of this format
is storing entire networks of data with respect to organised and labelled relationships - making it easy to search for data points
that belong to a particular class, or share a particular feature. The tutorial we used to understand SPARQL principles can be found here.
  Our Data Structure  
Where do we get our information?

The drug multi-functionality information we display on our Homepage is a product of two main sources: Wikidata and Bio2RDF. We
request different sets of data from each of these databases, the former being responsible for providing drugs relevant to a
disease cluster, and the latter providing clinical trial data corresponding to those drugs.

  Wikidata  
SPARQL Endpoint: https://query.wikidata.org/sparql

Wikidata is a large-scale source of information, stored in a format compatible with our queries. It stores "items", each with serialised
links and relationships to other "items" - an advantage it provides is a dedicated page to each item, providing the user with a visual
tabulation of the item's position in its network of data. You can find more on Wikidata here.

  Bio2RDF  
SPARQL Endpoint: https://bio2rdf.org/sparql

Like Wikidata, Bio2RDF stores data using RDF, but focuses more on biological data, containing information from literature and publications.
Its data is organised into 35 unique datasets, each with a specific focus. The database we make use of is clinicaltrails.gov, which
contains data about clinical studies around the world.
