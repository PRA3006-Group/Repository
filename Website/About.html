Programming in the Life Sciences

Programming in the Life Sciences is a skills course at the Maastricht Science Programme, Maastricht University.
The six-week course aims at giving students a brief introduction into SPARQL queries and their implementation
within web browsers using HTML and JavaScript.

The final course assignment includes a team-developed web page that utilises the concepts learnt throughout the
duration of the course. This website represents one such assignment made for the 2020-21 edition of the course
run by Egon Willighagen. More details can be found here and here
